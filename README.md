# FileDownloader
1) Download .zip and open FileDownloader.exe.
2) From the strart there will be two options - type "1" for file download or "2" for closing.
3) For download there will be first prompt for url adress of the image.
4) Next program will ask, if you want to choose directory, to which will be image saved (type "y" for yes, "n" for no).
5) If path does not exists, you will be asked, if you want to create this directory ("y" for yes, "n" for no).
6) If yes, file will be downloaded here, if no, you back on step 4.
